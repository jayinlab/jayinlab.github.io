title : fd 입문 가이드 - find보다 빠르게 파일 찾기
---

`fd`는 파일/폴더를 빠르게 찾는 도구다. `find`보다 문법이 직관적이라 초심자에게 특히 좋다.

## 언제 쓰면 좋은가
- 프로젝트에서 특정 이름의 파일을 찾을 때
- 확장자별 파일 목록을 빠르게 볼 때

## 자주 쓰는 예시
```bash
# 현재 폴더 아래에서 "config"가 들어간 파일/폴더 찾기
fdfind config

# md 파일만 찾기
fdfind -e md

# 숨김 파일까지 포함
fdfind -H env
```

## 초심자 팁
- Ubuntu에선 명령어가 `fd`가 아니라 `fdfind`일 수 있다.
- 먼저 `fdfind 키워드`만 써보고, 그다음 `-e`(확장자), `-H`(숨김 포함)를 붙여라.
- 결과가 너무 많으면 검색어를 더 구체적으로 적는 습관이 중요하다.

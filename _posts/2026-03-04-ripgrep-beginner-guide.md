`ripgrep`(`rg`)은 파일 **내용** 검색에 특화된 도구다. 매우 빠르고 기본값도 실용적이다.

## 언제 쓰면 좋은가
- "이 함수가 어디에서 쓰였지?"를 찾을 때
- 로그/문서에서 특정 문자열을 찾을 때

## 자주 쓰는 예시
```bash
# 현재 폴더에서 "TODO" 찾기
rg "TODO"

# 대소문자 무시 검색
rg -i "error"

# 특정 확장자만 검색
rg "useState" -g "*.tsx"
```

## 초심자 팁
- "파일 찾기"는 `fd`, "내용 찾기"는 `rg`로 구분하면 헷갈림이 줄어든다.
- 검색 결과가 많으면 먼저 `-g "*.확장자"`로 범위를 좁혀라.
- 정규식이 어렵다면 우선 일반 문자열 검색부터 시작해도 충분하다.

Platform
/ Device
clGetPlatformIDs()
clGetDeviceIDs()
GPU 작업 없음어떤 GPU를 쓸지 열거만 함
Context
clCreateContext()
GPU 작업 없음논리 영역 생성, 드라이버 상태만 초기화
Command
Queue
clCreateCommandQueue()
GPU 작업 없음명령 채널 준비, 아직 아무 명령도 없음
Program
clCreateProgramWithSource()
GPU 작업 없음소스 문자열 등록만, 컴파일 아직 안 함
Build
clBuildProgram()
컴파일clspv → SPIR-V → Vulkan 파이프라인 생성 ← 여기서 느림
Kernel
clCreateKernel()
clSetKernelArg()
CPU 전용실행 엔트리 + 인자 바인딩 (descriptor 준비)
Enqueue
clEnqueueNDRangeKernel()
큐 제출명령 큐에 등록 — 즉시 GPU 실행 아님!
Finish
clFinish()
동기화GPU 완료 대기 — 이 반환 후 결과 보장
